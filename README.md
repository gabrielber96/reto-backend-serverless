# RestApi The Star Wars API
## Usando Serverless Nodejs mysql

### 2 EndPoint (1 Get y 1 Post)
 - GET  /api/users/{id}
 - POST /api/users
### Npm usados
 - serverless-http
 - express
 - sequelize
 - mysql2
 - express-validator
 - serverless-offline (DEV)
 
 ### Uso
  #### modo local
  ```bash
  sls offline
  ```
  #### Deploy AWS
  ```bash
  sls deploy
  ```
